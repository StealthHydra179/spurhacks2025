# capyspend

Create .env file in root folder and follow this format:
```
PGHOST=localhost
PGUSER=postgres
PGPASSWORD=_____________________________
PGPORT=5432
PGDATABASE=spurhacks2025
JWT_SECRET=_____________________________
```
To run server
```bash
npm run server
```
